# E-Commerce Sales Database — SQL Task (Week 2)

This task was completly focussed on the basic sql queries about SELECT, WHERE, GROUP BY, JOINS, HAVING

## Sections

- **A — SQL Basics:** Basic SELECT queries, DISTINCT, primary keys, and constraints.
- **B — Filtering & Optimization:** WHERE filtering, date ranges, and how indexes / SARGable queries improve performance.
- **C — Aggregation:** COUNT, SUM, AVG, MIN, MAX with GROUP BY, ORDER BY, and HAVING.
- **D — Joins:** INNER, LEFT, and a 3-table join, plus foreign-key relationships and a FK-violation demo.
- **E — Advanced:** CASE for price tiers and in aggregates, the ACID properties, and a complete atomic transaction.
